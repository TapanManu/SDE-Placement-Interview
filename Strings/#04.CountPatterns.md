## #4. Count Patterns

```
Input: "ABCEABCABCD", "ABC"
Output: 3
```

**CPP**
```cpp
int containsPattern(string a, string b) {
	int n = a.length(), m = b.length();
	int i, j;

	int res = 0;

	for (i = 0; i < n - m + 1; ) {
		for (j = 0; j < m; j++) {
			cout << a[i + j] << " " << b[j] << endl;

			if (a[i + j] != b[j])
				break;
		}

		if (j == m)
			res++;

		if (j == 0)
			i++;
		else
			i += j;
	}

	return res;
}
```
